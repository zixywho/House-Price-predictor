# Bangalore House Price Predictor

A machine learning web application built with Streamlit that predicts Bangalore house prices based on:

- Location
- Total Square Feet
- Number of Bathrooms
- BHK

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit

## Features

- Interactive UI
- Real-time price prediction
- Trained Linear Regression/Ridge/Lasso model
- Location selection from Bangalore localities

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
